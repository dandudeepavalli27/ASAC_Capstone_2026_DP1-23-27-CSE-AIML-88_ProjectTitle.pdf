# ASAC_Capstone_2026_DP1-23-27-CSE-AIML-88_ProjectTitle.pdf
Smart Bin for Automatic Dry–Wet Waste Segregation using Arduino, ultrasonic sensor, moisture sensor, and servo motor for automated waste management.
Smart-Bin-Dry-Wet-Waste-Segregation

Software:
- Arduino IDE

Hardware:
- Arduino UNO
- Ultrasonic Sensor (HC-SR04)
- Soil Moisture Sensor
- Servo Motor (SG90)
- Breadboard
- Jumper Wires
- USB Cable

# Setup Instructions
1. Install Arduino IDE on your computer.
2. Connect the Arduino UNO board to the computer using a USB cable.
3. Connect the Ultrasonic Sensor (HC-SR04) to the Arduino (Trig → Pin 12, Echo → Pin 11).
4. Connect the Soil Moisture Sensor to Analog Pin A0.
5. Connect the Servo Motor to Pin 8.
6. Open the Arduino code from the src folder.
7. Upload the code to the Arduino UNO board.
8. Open the Serial Monitor in Arduino IDE.
9. Place waste near the sensor. The system will detect moisture and separate wet and dry waste automatically.
